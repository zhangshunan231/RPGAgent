# RPG节点编辑器

这是一个基于Unity GraphView的RPG生成器节点编辑器，提供了可视化的RPG内容生成界面。

## 功能特性

- **可视化节点编辑**：通过拖拽和连线的方式组织RPG生成流程
- **三种核心节点**：
  - 叙事节点：生成RPG故事和剧情
  - 场景节点：生成地图和场景参数
  - 机制节点：生成游戏机制和系统

## 使用方法

1. **打开编辑器**：在Unity菜单栏选择 `Tools > RPG Node Editor`

2. **添加节点**：
   - 点击工具栏中的"添加叙事节点"、"添加场景节点"、"添加机制节点"按钮
   - 节点会自动添加到画布中

3. **连接节点**：
   - 拖拽节点的输出端口到其他节点的输入端口
   - 数据会自动在连接的节点间传递

4. **配置参数**：
   - 在节点中填写相应的参数和设置
   - 场景节点可以配置地图参数和Tilemap引用

5. **执行生成**：
   - 点击工具栏中的"执行生成"按钮
   - 系统会按照节点连接顺序执行生成流程

## 节点类型说明

### 叙事节点 (NarrativeNode)
- **功能**：生成RPG故事和剧情
- **输入**：故事设想文本
- **输出**：结构化的叙事数据
- **特点**：通常作为工作流的起始节点

### 场景节点 (SceneNode)
- **功能**：生成地图和场景参数
- **输入**：叙事数据
- **输出**：场景配置数据
- **特点**：包含地图参数配置和Tilemap设置

### 机制节点 (MechanicsNode)
- **功能**：生成游戏机制和系统
- **输入**：叙事数据 + 场景数据
- **输出**：游戏机制数据
- **特点**：需要两个输入，通常作为工作流的终点

## 依赖项

- Unity 2021.3 或更高版本
- GraphView API (Unity内置)
- 现有的AgentCommunication系统
- 现有的地图生成系统

## 扩展开发

要添加新的节点类型：

1. 继承 `RPGNode` 类
2. 实现 `CreateInputPorts()`、`CreateOutputPorts()`、`CreateFields()` 方法
3. 在 `RPGNodeEditor.cs` 中添加创建新节点的按钮
4. 根据需要添加相应的样式

## 注意事项

- 确保Python Agent服务器正在运行
- 节点间的数据传递基于字符串类型
- 样式文件需要重新编译才能生效
